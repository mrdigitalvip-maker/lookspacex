# Roadmap do LookSpace

## Visão Geral

Este documento descreve o plano de desenvolvimento do LookSpace. As fases abaixo estão planejadas mas sujeitas a mudanças conforme novas prioridades e feedback emergem.

---

## Fase 0: Fundação (Concluído ✅)

- [x] Arquitetura base do projeto
- [x] Configuração do Next.js 14+ com App Router
- [x] TypeScript strict mode
- [x] Tailwind CSS com variáveis de cores espaciais
- [x] Integração Supabase
- [x] Estrutura de módulos
- [x] Documentação inicial

---

## Fase 1: Autenticação & Perfil

**Objetivo**: Usuários podem criar conta, fazer login e gerenciar perfil.

### Tasks
- [ ] Implementar tela de login
- [ ] Implementar tela de registro
- [ ] Sistema de recuperação de senha
- [ ] Perfil de usuário (edição de dados)
- [ ] Avatar do usuário
- [ ] Testes de autenticação

**Estimativa**: 2-3 semanas
